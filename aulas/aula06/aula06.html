Tratamento/Manipulação de Dados.
Comentários.
Comentáo em uma linha.

Para fazer um comentário de uma única linha basta colocar duas barras antes.

Ex:

// Esse é um exemploo de comentário de uma linha apenas.

Comentário em várias linhas ou comentário longo.

Para fazer um comentário de várias linhas, basta colocar, /*, no início da primeira linha e, */, no fim da última linha.

Ex:

/* Linha 1

Linha 2

Linha 3*/

Fixando a Quantidade de Dígitos a Direita.

> var n1 = 1545.5

1545.5

> n1.toFixed(3)

'1545.500'

Trocando a o Ponto por Vírgula.

> var n1 = 1545.5

1545.5

> n1.toFixed(2).replace('.', ',')

'1545,50'

Convertendo o número para o formato de moeda brasileira (R$)

> var n1 = 1545.5

1545.5

> n1.toLocaleString('pt-BR', {style:'currency', currency: 'BRL'})

'R$ 1,545.50'
